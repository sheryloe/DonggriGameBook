# CH06~CH20 챕터 시놉시스

## CH06 하강 관문

- 출발점: `CH05` 이후 판교 외곽 탈출 직후
- 목표: 남하 회랑에 진입할 수 있는 첫 통행권 확보
- 갈등: 판교 이탈민과 봉쇄선 잔존 인력이 동일한 출구를 두고 충돌
- 보스/위협: 검문소 주변 차량 잔해에 군집한 고착형 감염체
- 결과: 공식 통과선 또는 비공식 우회선 중 하나를 선택

## CH07 적색 회랑

- 목표: 붕괴된 고속 회랑을 통과해 남부 중계선 확보
- 갈등: 붉은 경고등만 남은 장거리 통로에서 추격과 매복이 반복
- 선택: 속도 우선 루트 vs 은밀 우회 루트
- 결과: 자원 소모와 감염 위험의 균형이 다음 장에 영향

## CH08 봉쇄선의 방

- 목표: 봉쇄선 행정 통제실 침투
- 갈등: 통행증 체계가 아직 자동 검문 시스템과 연결되어 있음
- 선택: 문서 위조, 전력 차단, 내부 협력자 매수
- 결과: 회랑 공식 데이터 또는 내부 배신자 명단 확보

## CH09 연기 저장고

- 목표: 장거리 이동에 필요한 연료/배터리 확보
- 갈등: 화재와 유독성 연기 때문에 시간 압박이 심함
- 선택: 위험한 대량 보급 vs 안전한 소량 보급
- 결과: CH10 항만 접근 난이도와 적재량이 달라짐

## CH10 침하 항만

- 목표: 동해 접근용 선박 또는 해상 운송 계약 확보
- 갈등: 침하한 화물 부두에서 해상 루트를 놓고 세력 충돌
- 선택: 계약, 강탈, 구조 협상
- 결과: CH11 진입 조건인 외부 접근권 획득

## CH11 철의 우회

- 목표: 철도 우회선을 통해 내륙 잔존 구간을 통과
- 갈등: 선로 정비권을 장악한 집단과의 거래
- 선택: 선로 복구 협력 vs 방치된 화물차량 분해
- 결과: 이동 시간 단축 또는 전투 리스크 증가

## CH12 잔향 기지

- 목표: 군 전파 기지의 마지막 로그 해독
- 갈등: 명령 체계는 죽었지만 보안 장치는 살아 있음
- 선택: 기록 회수 우선 vs 송신기 복구 우선
- 결과: 해안 접근 좌표와 감염 주파수 단서 획득

## CH13 백색 야적장

- 목표: 냉동 야적장에서 의약품과 냉각 코어 회수
- 갈등: 시체 물류 흔적이 드러나며 생존 세력 신뢰가 흔들림
- 선택: 의약품 회수 vs 증거 확보
- 결과: NPC 신뢰도와 최종 루트 윤리 판단에 영향

## CH14 해무 변전소

- 목표: 해안 접근 전력망 재기동
- 갈등: 해무와 부식 환경 때문에 시야/장비 손실이 큼
- 선택: 변전소 전체 복구 vs 일부 라인만 응급 사용
- 결과: CH15 외해 전초 접근 난이도 변화

## CH15 격리 파수

- 목표: 외해 전초 앞 검문선 붕괴를 돌파
- 갈등: 잔류 경비 세력이 여전히 선별 규칙을 집행
- 선택: 강행 돌파, 정식 인증, 내부 협력
- 결과: CH16 시작 상태를 결정

## CH16 균열 사구

- 목표: 해안 붕괴 구간을 넘어 외해 출발선 유지
- 갈등: 이동 경로가 지형 자체와 싸우는 구조
- 선택: 느리지만 안전한 우회 vs 빠르지만 붕괴 위험한 직선
- 결과: 보급량과 동료 상태가 흔들림

## CH17 파편 수문

- 목표: 해안 수문 회랑을 열어 외해 장비 이동
- 갈등: 염분과 부식으로 장비 사용 수명이 급격히 감소
- 선택: 수문 강제 개방 vs 순차 해제
- 결과: CH18 자원 압박 변화

## CH18 소금 정거장

- 목표: 최종 보급과 승선 전 점검
- 갈등: 남은 자원을 누구에게 배분할지 결정해야 함
- 선택: 자신, 동료, 플랫폼 생존자 사이의 배분
- 결과: CH19 지원 세력 규모가 달라짐

## CH19 외해 전초

- 목표: 독도 접근용 외해 플랫폼 확보
- 갈등: 접근 인증과 승선 우선권을 둘러싼 대립
- 선택: 선별 체계 수용 vs 체계 붕괴 시도
- 결과: CH20 진입 루트와 최종 엔딩 분기 결정

## CH20 독도의 문

- 목표: 독도 진입 또는 진입 거부의 최종 선택
- 갈등: 독도가 피난처인지, 마지막 선별 장치인지 확인
- 선택: 독도 진입, 독도 봉인, 외해 잔류
- 결과: 캠페인 엔딩 확정
