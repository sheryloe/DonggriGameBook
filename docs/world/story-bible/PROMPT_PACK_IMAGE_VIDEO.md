# PROMPT PACK IMAGE VIDEO

## 기본 원칙

- 미감은 `한국 도시 재난물 리얼리즘 + 시네마틱 압축`이다.
- 캐릭터와 허브는 사실감 우선, 보스와 티저는 대비와 스케일을 한 단계 올린다.
- 프롬프트는 영어 중심 생성 구문에 한국어 맥락 키워드를 보조로 섞는다.
- 실제 상표, 게임 로고, 유명 캐릭터 유사성은 금지한다.

## 공통 키워드

- `Korean urban apocalypse`
- `documentary disaster realism`
- `wet concrete`
- `patched quarantine signage`
- `civic survival bureaucracy`
- `salt corrosion`
- `radio static atmosphere`
- `practical clothing, no tactical cosplay`

## 공통 네거티브

- `no superhero pose`
- `no cyberpunk neon city`
- `no glossy sci-fi armor`
- `no recognizable brands`
- `no excessive gore`
- `no anime proportions`

```text

```

## 영상 프롬프트 템플릿

### 1. 15초 챕터 티저

```text
15-second cinematic teaser, Korean apocalypse chapter [chapter title], quick cuts of [location 1], [location 2], [signature threat], wet concrete, warning lights, radio static transitions, realistic camera movement, tense breathing, no dialogue subtitles, end on cliffhanger image
```

### 2. 45초 파트 시네마틱

```text
45-second cinematic trailer for Part [part id] of a Korean apocalypse gamebook, show [hub], [two travel locations], [two recurring characters], [signature boss], documentary realism mixed with cinematic tension, radio broadcast fragments, survival bureaucracy details, final shot with unresolved hope
```

## 실무 메모

- 캐릭터 프롬프트는 `직업 + 감정 + 장비 1개 + 배경 1개`를 반드시 포함한다.
- 허브 프롬프트는 `운영 흔적`, `배급 흔적`, `검역 흔적`을 동시에 포함한다.
- 보스 프롬프트는 `장소 기능`과 `공략 포인트`가 읽히도록 작성한다.
- 영상 프롬프트는 컷 수보다 `갈등 1개`, `장소 2개`, `위협 1개`를 명확히 적는 편이 안정적이다.
