# CANON_APPENDIX_01_20

## 용도

- 이 문서는 `플레이어용 본문`이 아니라 `공식 연속성 정리`용 부록이다.
- 본편 문서에는 정사를 직접 못 박지 않는다.
- 후속 파트, 외전, 마케팅 문구, 트레일러 문장 정렬이 필요할 때만 이 문서를 쓴다.

## 공식 정사 선택

- 공식 정사: `기록 공개 후 제한적 재설계`
- 이유:
  - `선별 유지`는 세계관 긴장을 보존하지만 주제상 회복이 약하다.
  - `선별 파괴`는 카타르시스는 크지만 후속 재건의 기반이 너무 얇다.
  - `기록 공개 후 제한적 재설계`는 희망과 대가, 기록과 통제, 공개와 책임을 모두 남길 수 있다.

## 정사 엔딩 요약

1. 윤해인은 기록을 전면 공개한다.
2. 김아라는 `capacity_slot` 사전 선별 방식을 폐기하고 `공개된 triage + 실시간 수용량`으로 코어 규칙을 수정한다.
3. 차문식의 보존 논리는 패배하지만, 그가 경고한 `수용량의 한계` 자체는 사라지지 않는다.
4. 독도는 낙원이 아니라 `재건 가능한 최소 체계`로 남는다.
5. 바깥 생존권은 넓어지지만, 누군가는 남고 누군가는 늦게 들어와야 한다는 대가는 유지된다.

## 정사 후 주요 인물 상태

| 인물 | 정사 상태 |
|---|---|
| 윤해인 | 최종 방송 송출 후 생존. 기록 공개파의 실질 리더 |
| 정노아 | 위조망 지식을 공개 전환해 인증 우회 기록과 통로를 증언 |
| 서진서 | 외해 이동과 보급선을 책임지는 항로 관리자 역할로 이동 |
| 안보경 | CH15 희생으로 사망. CH16 이후 파티 운영 규칙을 바꾸는 부재로 남음 |
| 류세온 | 후반에 선별 유지 논리를 끝까지 주장하다 패배 또는 퇴장 |
| 한소명 | 공개된 triage 체계의 배분 규칙 설계에 참여 |
| 김아라 | 코어 개조 작업에 남거나 장기 잔류. 엔딩별 생존은 열려 있으나 정사에서는 잔류 생존 |
| 차문식 | 공개 토론과 플랫폼 장악전에서 패배. 기록 속 관리자의 얼굴로 고정 |

## 후속 연속성 메모

- 후속 문서에서는 `독도는 열린 천국`이 아니라 `좁지만 공개된 입구`로 묘사한다.
- 정사 이후에도 `health_clear`, `route_code`, `capacity_slot`은 완전히 사라지지 않는다.
- 사라지는 것은 `비공개 사전 선별`이며, 남는 것은 `공개된 수용량 관리`다.
