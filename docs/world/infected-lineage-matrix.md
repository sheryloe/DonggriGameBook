# 감염체 계보표

## 원칙

- 파트가 나뉘어도 감염체는 세계관상 같은 계보를 공유한다.
- 챕터별 보스는 지역 장치와 결합한 변종으로 설계한다.

## 계열

### 1. 부착형

- 특징: 구조물에 고정되거나 장치와 융합
- 사용 챕터:
  - CH01 편집실/송출 장치
  - CH05 미러센터 코어
  - CH17 수문 회랑

### 2. 수로형

- 특징: 물, 배수, 진흙, 하수와 결합
- 사용 챕터:
  - CH02 노량진 수로
  - CH10 침하 항만
  - CH16 해안 붕괴선

### 3. 수직 추적형

- 특징: 계단, 사다리, 엘리베이터 샤프트에서 공포 형성
- 사용 챕터:
  - CH03 유리정원
  - CH14 변전소 상부
  - CH19 외해 플랫폼

### 4. 물류형

- 특징: 컨베이어, 포장, 운송, 냉동 적재와 결합
- 사용 챕터:
  - CH04 상자들의 도시
  - CH09 연기 저장고
  - CH13 백색 야적장

### 5. 선별 잔재형

- 특징: 문서, 신호, 인증 절차와 함께 등장하는 후기 변종
- 사용 챕터:
  - CH05 미러센터
  - CH12 잔향 기지
  - CH20 독도의 문

## 구현 규칙

- 같은 계열은 애니메이션 리그와 사운드 규칙을 최대한 재사용
- 보스는 계열 1개 + 지역 장치 1개를 조합해 정의
- CH06~CH20은 신규 계열 추가보다 기존 계열 변주를 우선
