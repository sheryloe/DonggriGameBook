# Story 02 Asset Prompts

## 공통 규칙

- 세계관은 `여의도 외곽 보급 루트`와 `검문-교환-복귀` 기반이다.
- 톤은 `한국형 생존 관료주의 + 외곽 회수 경제 + 젖은 콘크리트`로 맞춘다.
- 공식 캐릭터나 로고 직접 재현 금지.
- 화면은 문서, 도식, 영수증, 배차표, 검문표처럼 보여야 한다.
- 기존 Story 02 콜라주 자산을 재사용하고, 신규 파일은 만들지 않는다.

## Node Prompt Map

### 1. Dispatch

`Korean survival dispatch board for Yeouido outer supply route, name card, cargo manifest, thermal scan strip, route number, bureaucratic logistics collage, damp paper and alarm lights`

### 2. Floodwall

`Yeouido floodwall route schematic, river embankment, service road, wet concrete, emergency lights, Korean checkpoint signage, documentary logistics infographic`

### 3. Tollgate

`Outer tollgate checkpoint dashboard, vehicle barrier, manifest scan, access card check, route clearance paperwork, narrow traffic bottleneck, no characters`

### 4. Market

`Under-bridge barter market infographic, water jugs, battery exchange boxes, transport cards, ration crates, handwritten price marks, Korean survival economy texture`

### 5. Relay

`Supply relay warehouse board, redistribution ledger, resident request slips, cargo stacks, official and unofficial processing mix, Korean post-collapse bureaucracy`

### 6. Sluice

`Yeouido floodgate and service culvert schematic, water level gauge, barrier lights, maintenance route, wet concrete, route pressure infographic, no characters`

### 7. Depot

`Temporary cargo depot board, pallet stacks, inventory seal tape, forklift tracks, crate tally sheet, reflective cones, Korean relief logistics texture`

### 8. Return

`Checkpoint return sheet for Yeouido supply route, approval and debt stamps, witness log, green scan light, official form tone`

## Endings

- `ending-clear`: `Approved supply-route ending card, green stamp, recovered logistics line, quiet success`
- `ending-debt`: `Amber debt return card, cargo deduction, monitoring tag, conditional approval`
- `ending-quarantine`: `Red quarantine failure card, scanline texture, route contamination warning, hold status`

## Prompt Notes

- `sluice` should feel like a pressure point, not an action scene.
- `depot` should feel like inventory, seal, and omission, not a loot room.
- Use more bridge, floodwall, cargo, and exchange imagery than interior shelter imagery.
- Prefer concrete, service-road, and market detail over action-heavy or military-heavy imagery.
