# ARC 01~05 설계 팩
이 문서는 웹게임/스토리북 기반 한국형 회수 아포칼립스 프로젝트의 **Chapter 1~5 상세 설계 초안**이다.

## 반경 구성
- Chapter 1: 여의도 기록국 / 방송권
- Chapter 2: 노량진 수몰 상권 / 동작 접근선
- Chapter 3: 잠실 초고층 주거단지
- Chapter 4: 문정-가락-탄천 물류벨트
- Chapter 5: 판교 테크노밸리 재난 미러센터

## 상표/실재 시설 리스크 대응
사용자가 제안한 **'잠실 롯데캐슬'**은 분위기만 살리고, 문서상에서는 **'잠실 캐슬원 레지던스'**로 픽션화했다.  
실제 장소감은 유지하되, 고유 브랜드 및 1:1 식별 요소는 피하는 방향이다.

## 공통 설계 원칙
1. **루팅 = 서사 행위**  
   탄약보다 문서, 배터리, 필터, 인증카드가 더 중요하게 작동한다.

2. **허브 ↔ 원정 ↔ 귀환 구조**  
   각 챕터는 한 번 깨고 끝나는 선형 구간이 아니라, 이후 재진입할 이유가 남는다.

3. **선택지는 감정이 아니라 시스템도 바꾼다**  
   생존자 수, 신뢰도, 장비 해금, 다음 챕터 시작 자원, 보스 패턴까지 영향을 준다.

4. **감염체는 지역 환경과 연결된다**  
   수몰 지역, 고층 주거지, 물류 자동화 구역, 서버실 등 공간 논리가 곧 적 디자인 논리다.
